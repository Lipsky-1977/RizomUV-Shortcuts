# RizomUV 2025 快捷键查询

一个纯 HTML 快捷键查询工具，**无需安装任何依赖**，手机和 PC 浏览器均可直接打开使用。

## 使用方式

直接打开 `index.html` 即可，或将文件传到手机浏览器中。

## 功能

- **110 条快捷键**，覆盖 10 个常用分类
- **实时搜索**：按快捷键名称、按键组合或功能描述筛选
- **分类筛选**：视图控制、选择操作、UV变换、UV展开与打包、显示与纹理等
- **深色模式**：跟随系统主题自动切换
- **触控优化**：移动端单手操作适配
- **离线可用**：单文件，不依赖网络

## 快捷键分类

| 分类 | 说明 |
|------|------|
| 视图控制 | 视图环绕/平移/缩放、网格切换 |
| 选择操作 | 选择/框选、孤岛选择、隐藏显示 |
| UV变换 | 移动/旋转/缩放UV、翻转镜像 |
| UV展开与打包 | Unfold展开、Pack打包、松弛堆叠 |
| 显示与纹理 | 纹理/棋盘格、线框、形变图 |
| 锁定与约束 | Pin锁定、Unpin解锁、隔离模式 |
| 编辑操作 | 撤销、删除、模式切换 |
| 文件操作 | 新建、打开、保存、导出UV |
| 工具 | 帮助、设置、搜索、配置 |
| 选择模式 | 孤岛/多边形/边/边界/环模式 |
